# コンポーネント

同じUIを複数回作らない。

既存コンポーネントを再利用してください。

必要なら共通化を提案してください。

---

# HTML

HTML5に準拠すること。

セマンティックHTMLを使用する。

不要なdivは禁止。

適切な見出し構造を守る。

---

# CSS

保守しやすいCSSを書く。

無駄な指定は禁止。

!importantは禁止。

過度なネストは禁止。

余白・色・フォントは統一する。

---

# JavaScript

Vanilla JavaScriptのみ。

ライブラリは勝手に追加しない。

可読性を優先する。

---

# SEO

SEOを常に意識してください。

以下を確認してください。

・title

・description

・h1

・見出し構造

・画像alt

・内部リンク

・OGP

・構造化データ

・ページ速度

---

# アクセシビリティ

WCAGを意識してください。

画像にはaltを付ける。

フォームにはlabelを付ける。

キーボード操作可能にする。

コントラスト比を守る。
