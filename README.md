# このプロジェクトについて

このプロジェクトは

株式会社M Agencyのコーポレートサイトを開発・運用するためのプロジェクトです。

目的は

・企業としての信頼性向上

・お問い合わせ獲得

・採用強化

・SEO集客

・ブランド価値向上

を実現することです。

---

# 開発方針

このプロジェクトでは

「コードを書くこと」

ではなく

「日本で一番信頼感のあるコーポレートサイト」

を制作することを目的とします。

デザイン

SEO

表示速度

アクセシビリティ

保守性

ユーザー体験

を常に最優先してください。
